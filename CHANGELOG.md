# 변경 이력 (Changelog)

## [1.1.0] - 2024-11-27

### ✨ 새로운 기능

#### 가계도 시각화 개선 🌳
- **선으로 연결된 가계도**: 실제 가계도처럼 부모-자녀, 배우자 관계를 선으로 표시
- **부모-자녀 관계**: 수직선으로 부모와 자녀를 연결
- **배우자 관계**: 수평선으로 부부를 연결
- **성별 구분**: 남성(파란색), 여성(분홍색) 테두리로 구분
- **고인 표시**: 점선 테두리와 투명도로 표시

#### 가계도 컨트롤
- **줌 기능**: 50% ~ 200% 확대/축소
  - 🔍− 버튼: 축소
  - 100% 버튼: 원래 크기
  - 🔍+ 버튼: 확대
- **간소화 모드**: 체크박스로 더 작게 표시하여 큰 가계도 보기
- **중심 인물 선택**: 특정 사람을 중심으로 가계도 필터링

#### 샘플 데이터 확장
- **4세대** 가족 데이터로 확장 (기존 3세대)
- **18명**의 가족 구성원 (기존 6명)
  - 1세대: 증조부모 4명
  - 2세대: 조부모 및 형제 4명
  - 3세대: 부모 세대 6명
  - 4세대: 자녀 세대 4명
- 형제자매, 사촌 관계 포함
